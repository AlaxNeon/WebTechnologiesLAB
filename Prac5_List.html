Demonstration of Lists in HTML

Ordered Lists (Numbered Lists)

Default Ordered Lists:

  1. Apple
  2. Banana
  3. Cherry

Ordered List with type = "A" (Uppercase):

  1. Item One
  2. Item Two
  3. Item Three

Ordered List with type = "a" (Lowercase):

  1. First
  2. Second
  3. Third

Ordered List with type = "I" (Uppercase Roman Numerals):

  1. Chapter One
  2. Chapter Two
  3. Chapter Three

Ordered List with type = "i" (Lowercase Roman Numerals):

  1. Step One
  2. Step Two
  3. Step Three

Ordered List with start = "5":

  1. Alpha
  2. Beta
  3. Gamma

Ordered List with reversed numbering:

  1. Point A
  2. Point B
  3. Point C
